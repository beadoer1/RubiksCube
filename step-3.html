루빅스 큐브 구현하기

아래 버튼을 눌러 조작 하거나 명령어를 입력하여 큐브를 맞추어 보세요.

※ 동작 방법
T(Top) : 윗면 왼쪽 돌리기, Bo(Bottom) : 아랫면 오른쪽 돌리기, L(Left) : 왼면 아래로 밀기, R(Right) : 오른면 위로 밀기, F(Front) : 앞면 오른쪽 돌리기, Ba(Back) : 뒷면 왼쪽 돌리기, Q(Quit) : 게임 종료
T' : 윗면 오른쪽 돌리기, Bo' : 아랫면 왼쪽 돌리기, L' : 왼면 위로 밀기, R' : 오른면 아래로 밀기, F' : 앞면 왼쪽 돌리기, Ba' : 뒷면 왼쪽으로 돌리기
